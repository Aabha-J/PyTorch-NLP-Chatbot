After downloading the files or cloning this repository do these 
steps in the terminal to see the chatbot in action

To install nesseacry libraries
1. Set up a virtual python enviorment
2. pip install numpy streamlit torch nltk

To run chat in terminal:
python run chat.py

To run chat with streamlit UI:
streamlist run app.py


If your processor is slow,use the Chatbot_in_Colab file in google colab to train the model
and then download it. or use the provided data.pth file
